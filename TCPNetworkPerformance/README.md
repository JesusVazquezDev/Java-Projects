# Lab1: Sockets, Mininet & Performance

A comprehensive exploration of network programming, socket communication, and performance analysis using Mininet simulation environment.

## 📋 Overview

This lab focuses on understanding TCP socket behavior, network performance optimization, and the practical application of networking concepts in a controlled simulation environment.

## 🎯 Objectives

- Implement socket-based network communication
- Analyze TCP performance characteristics
- Optimize network communication patterns
- Profile and benchmark socket operations
- Understand network bottlenecks and optimization strategies

## 🛠️ Technologies & Tools

- **Language**: Java
- **Networking**: TCP/IP Sockets
- **Simulation**: Mininet
- **Performance Analysis**: Profiling, Benchmarking

## 📊 Key Experiments

1. **Basic Socket Communication** - Foundation of TCP/IP connectivity
2. **Performance Measurement** - Throughput and latency analysis
3. **Optimization Techniques** - Buffer sizing, protocol tuning
4. **Comparative Analysis** - Different configuration impact assessment
